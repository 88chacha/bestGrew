# 우수수료생 선정 도구 — Plan

## 개요

Google Sheets에 저장된 학습자 데이터를 읽어, 퀘스트 점수·노드 별점·비정규 점수·성실 점수의 반영 비율을 실시간으로 조정하면서 과정+기수별 우수수료생을 선정하는 Streamlit 대시보드.

## 점수 구성

### 1. 퀘스트 점수 (`퀘스트합계` = 일반 퀘스트 + Main QUEST)
- **일반 QUEST** (QUEST_XX, Sub Quest XX, QuestB_XX 등): `P = 1점`, `F / null / 미제출 = 0점`
- **Main QUEST** ([PROJECT] Main 포함): 셀의 숫자값 그대로 반영

### 2. 노드 별점 (`노드별점`)
- `노드퀘스트DB`에서 `(고유번호, 이름)` 조합 기준으로 학습자별 별점 합산
  - 고유번호 중복 할당 케이스 대응: 이름으로 추가 구분
- 유효 범위: 0~3점

### 3. 비정규 점수 (`비정규합계`)
- 구성 항목 (퀘스트 시트에서 읽음):
  - 디스코드 소통왕
  - 아낌없이 주는 그루
  - 쉐밸그투
  - 퍼실재량점수

### 4. 성실 점수 (`성실점수`)
- **퀘스트 제출율**: 전체 퀘스트(일반+Main) 중 제출한 항목 비율
  - 제출 = P, F, 숫자값 / 미제출 = null, 미제출, 미참여, 중도포기, 빈값
- **노드 제출율**: `노드퀘스트DB` 기준, 별점이 있는 고유 노드 수 / 전체 고유 노드 수
- **성실점수** = (퀘스트 제출율 + 노드 제출율) / 2
- 이미 0~100 범위이므로 추가 정규화 없이 그대로 사용

## 점수 계산 방식

```
종합점수 = (퀘스트_정규화 × W퀘스트 + 별점_정규화 × W별점 + 비정규_정규화 × W비정규 + 성실_정규화 × W성실)
           / (W퀘스트 + W별점 + W비정규 + W성실)
```

### 정규화 방법

| 항목 | 정규화 방식 |
|------|------------|
| 퀘스트 점수 | 기수별 만점 기준: `원점수 / 만점 × 100` (100 초과 클리핑) |
| 노드 별점 | 기수별 만점 기준: `원점수 / 만점 × 100` (100 초과 클리핑) |
| 비정규 점수 | 과정+기수 그룹 내 Min-Max: `(원점수 - 최솟값) / (최댓값 - 최솟값) × 100` |
| 성실 점수 | 정규화 없음 (제출율 자체가 0~100) |

- 만점 미등록 기수(엔지니어, 리서치 17기 등)는 퀘스트·별점도 그룹 내 Min-Max로 fallback
- 그룹 내 인원 1명이거나 최댓값=최솟값이면 50점 고정

### 기수 추출 기준
- 항상 시트명에서 추출 (노드퀘스트DB의 기수 컬럼은 "6기" 형태로 prefix 없음)
- 예: `퀘스트(데싸6기)` → `데싸6기`, `퀘스트(7기코어)` → `7기`

## UI 구성 (`app.py`)

### 사이드바

| 컨트롤 | 설명 |
|--------|------|
| 퀘스트 점수 슬라이더 | 반영 비율 (0~100, 기본 40) |
| 노드 별점 슬라이더 | 반영 비율 (0~100, 기본 25) |
| 비정규 점수 슬라이더 | 반영 비율 (0~100, 기본 15) |
| 성실 점수 슬라이더 | 반영 비율 (0~100, 기본 20) |
| 기준 점수 슬라이더 | 종합점수 기준선 (0.0~100.0), 이 값 이상이면 우수수료생 |
| 과정 선택 | 전체 / 코어 / 리서치 / DS / PDA / 엔지니어 |
| 훈련상태 필터 | 정상수료·훈련중 등 복수 선택 |

> 4개 비율의 합계로 자동 정규화 (합계가 100이 아니어도 동작)

### 탭 구성

#### Tab 1 — 기수별 우수수료생
- 과정별 섹션 → 기수별 펼침 패널
- 각 패널: 전체 N명 / 우수수료생 M명 / 선정률 + 막대 차트
- 우수수료생 명단 테이블: 원점수 + 정규화 점수 함께 표시
  - 퀘스트(원점수/정규화), 별점(원점수/정규화), 비정규(원점수/정규화), 퀘스트제출율(%), 노드제출율(%), 성실(정규화), 종합점수
- 비정규 점수 항목별 상세 (펼침)

#### Tab 2 — 전체 순위표
- 과정·기수·순위·이름·훈련상태·모든 점수 항목 테이블
- 원점수 + 정규화 점수 + 성실 점수 포함
- 우수수료생 행 초록 하이라이트

#### Tab 3 — 과정별 요약
- 과정별 우수수료생 수 / 선정률 막대 차트
- 퀘스트·별점·비정규 평균·중앙값 요약 테이블
- 점수 분포 박스플롯 (과정 × 항목)
- 항목별 정규화 평균 비교 막대 (과정별)

#### Tab 4 — 점수 분포
- 종합점수 히스토그램 (우수수료생/일반 색 구분, 기준선 표시)
- 퀘스트 vs 별점 산점도 (원 크기=비정규 점수)

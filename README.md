# 우수수료생 선정 도구

Google Sheets에 저장된 학습자 데이터를 기반으로, 점수 반영 비율을 실시간으로 조정하며 과정·기수별 우수수료생을 선정하는 Streamlit 대시보드.

---

## 주요 기능

- **점수 가중치 실시간 조정**: 퀘스트·노드 별점·비정규·성실 점수의 반영 비율을 슬라이더로 조정
- **과정+기수별 선정**: 코어·리서치·DS·PDA·엔지니어 과정별, 기수별로 우수수료생 독립 선정
- **상위 15% 추가 추천**: 우수수료생 선정 비율이 15% 미만인 기수에서 자동으로 추가 추천 인원 표시
- **원점수 + 정규화 점수 동시 표시**: 두 점수를 함께 확인하여 직관적인 비교 가능
- **4개 탭 뷰**: 기수별 우수수료생 / 전체 순위표 / 과정별 요약 / 점수 분포

---

## 점수 구성

| 항목 | 설명 |
|------|------|
| 퀘스트 점수 | 일반 QUEST (P=1점, F/미제출=0점) + Main QUEST (숫자값) |
| 노드 별점 | 노드퀘스트DB 기준, `(고유번호, 이름)` 조합으로 별점 합산 |
| 비정규 점수 | 디스코드 소통왕 + 아낌없이 주는 그루 + 쉐밸그투 + 퍼실재량점수 |
| 성실 점수 | (퀘스트 제출율 + 노드 제출율) / 2 |

### 정규화 방식

| 항목 | 방식 |
|------|------|
| 퀘스트·노드 별점 | 기수별 만점 기준 정규화 (`원점수 / 만점 × 100`) |
| 비정규 점수 | IQR 적응형 윈저라이징 후 Min-Max (이상치 없으면 clip 무효) |
| 성실 점수 | 정규화 없음 (0~100 그대로 사용) |


---

## 데이터 소스

- **Google Sheets** (Sheets API v4, 서비스 계정 인증)
- `노드퀘스트DB` 시트: 학습자별 노드 별점 원천 데이터
- `퀘스트(XX기YY)` 시트: 기수+과정별 퀘스트 및 비정규 점수

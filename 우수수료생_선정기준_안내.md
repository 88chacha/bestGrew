# 우수수료생 선정 기준(안) — 점수 산출 방식 안내

## 1. 배경

우수수료생 선정을 위해서는 과정과 기수가 서로 다른 학습자들을 **동일한 잣대**로 평가할 수 있어야 합니다. 그러나 기수마다 퀘스트 개수, 노드 수, 비정규 활동 기회가 다르기 때문에 원점수를 그대로 비교하면 공정한 평가가 어렵습니다.

예를 들어:
- 코어 6기의 퀘스트 만점은 84점이지만, 코어 7기는 90점
- 리서치 5기의 노드 만점은 57점이지만, 리서치 8기는 48점
- 비정규 점수는 기수마다 활동 기회와 편차가 크게 다름

이 차이를 보정하지 않으면 만점이 높은 기수의 학습자가 유리해지거나, 특정 기수에서 이상치(극단적 고점자) 1명 때문에 나머지 학습자의 점수가 과소 평가되는 문제가 발생합니다.

---

## 2. 정규화가 필요한 이유

| 문제 상황 | 원점수만 쓸 때 발생하는 문제 | 정규화 적용 시 |
|-----------|---------------------------|---------------|
| 기수별 만점 차이 | 퀘스트 만점 84점 기수와 93점 기수를 동일 기준으로 비교 불가 | 각 기수 만점 대비 비율로 통일 (0~100점) |
| 비정규 점수 이상치 | 1명이 비정상적 고점 → 나머지 전원의 비정규 점수가 0에 수렴 | 통계적 이상치만 보정, 정상 분포는 변별력 유지 |
| 항목 간 스케일 차이 | 퀘스트(0~93점) vs 노드(0~57점) vs 비정규(0~20점대) → 퀘스트가 과도하게 지배 | 모두 0~100으로 통일 후 가중 합산 |

---

## 3. 항목별 정규화 방식

### 3-1. 퀘스트 점수 / 노드 별점: 만점 기준 정규화

```
정규화 점수 = (원점수 / 해당 기수 만점) × 100
```

- 각 기수의 공식 만점이 확인되어 있으므로, 만점 대비 달성 비율로 환산
- 100점 초과 시 100점으로 클리핑
- 가장 직관적이고 공정한 방식 (80점/100점 만점 → 80% 달성)

**왜 Min-Max가 아닌 만점 기준인가?**
- Min-Max는 해당 기수 내 최고점자를 100, 최저점자를 0으로 만들어 **절대적 성취도를 반영하지 못함**
- 만점 기준 정규화는 "기수 만점 대비 얼마나 달성했는가"라는 **절대적 기준**을 유지

### 3-2. 비정규 점수: IQR 적응형 윈저라이징 + Min-Max

```
1) 이상치 기준선 = Q3 + 1.5 × IQR (사분위 범위)
2) 이상치 기준선 초과 점수만 기준선으로 보정
3) 보정된 점수로 Min-Max 정규화 (0~100)
```

비정규 점수는 기수마다 공식 만점이 정해져 있지 않아 만점 기준 정규화가 불가능합니다. 대신 그룹 내 상대적 위치(Min-Max)를 사용하되, 이상치 문제를 해결하기 위해 **IQR 적응형 윈저라이징**을 먼저 적용합니다.

**핵심 특징:**
- 이상치가 **없는** 기수: 보정 없이 일반 Min-Max와 동일하게 동작 → 변별력 유지
- 이상치가 **있는** 기수: 극단적 고점만 통계적으로 보정 → 나머지 학습자의 과소 평가 방지
- IQR 1.5배 기준은 통계학에서 가장 널리 쓰이는 이상치 판별 기준 (Tukey's Fences)

### 3-3. 성실 점수: 정규화 없음

```
성실점수 = (퀘스트 제출율 + 노드 제출율) / 2
```

- 제출율 자체가 이미 0~100% 범위
- 기수 간 스케일 차이가 없으므로 추가 정규화 불필요

---

## 4. 종합 점수 산출

```
종합점수 = (퀘스트_정규화 × W1 + 별점_정규화 × W2 + 비정규_정규화 × W3 + 성실_정규화 × W4)
           / (W1 + W2 + W3 + W4)
```

### 기본 가중치(안)

| 항목 | 가중치 | 비율 | 근거 |
|------|:------:|:----:|------|
| 퀘스트 점수 | 50 | 50% | 핵심 학습 성과를 가장 직접적으로 반영 |
| 노드 별점 | 20 | 20% | 학습 품질 (퍼실리테이터 평가) 반영 |
| 비정규 점수 | 20 | 20% | 커뮤니티 기여도 반영 |
| 성실 점수 | 10 | 10% | 꾸준한 참여 태도 반영 |

> 가중치는 대시보드에서 실시간 조정 가능하며, 합계가 100이 아니어도 비율로 자동 환산됩니다.

---

## 5. 우수수료생 선정 기준

- **기준 점수**: 종합점수 **80점 이상** (기본값, 조정 가능)
- **선정 단위**: 과정 + 기수별 독립 선정
- **추가 추천**: 선정 인원이 기수 전체의 15% 미만일 경우, 종합점수 상위 15% 인원을 별도 추천

---

## 6. 정리

| 단계 | 내용 |
|------|------|
| 1단계 | 기수별 원점수 수집 (퀘스트, 노드 별점, 비정규, 성실) |
| 2단계 | 항목별 정규화로 기수 간 스케일 차이 해소 (0~100점 통일) |
| 3단계 | 가중치 적용하여 종합점수 산출 |
| 4단계 | 기준 점수 이상인 학습자를 과정+기수별로 우수수료생 선정 |
| 5단계 | 선정률이 낮은 기수에 대해 상위 15% 추가 추천 |

> 모든 수치는 대시보드에서 실시간으로 조정하며 결과를 확인할 수 있습니다.
